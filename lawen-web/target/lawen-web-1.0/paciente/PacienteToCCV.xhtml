En esta sección podrá ingresar atenciones del programa de salud cardiovascular.

Para crear un preingreso, debe seleccionar cual preingreso quiere ingresar, y luego hacer clic en crear preingreso.
Para crear un ingreso, debe seleccionar cual ingreso quiere registrar, y luego hacer clic en crear ingreso.
Para crear un control cardiovascular, debe seleccionar cual control quiere ingresar, y luego hacer clic en crear control.

Crear preingreso I, II y III

Para crear un preingreso del programa de salud cardiovascular, necesita por lo menos llenar los siguientes campos:

Tipo de programa - Fecha - Profesional

Crear ingreso Médico y Nutricionista

Para crear un ingreso del programa de salud cardiovascular, necesita por lo menos llenar los siguientes campos:

Fecha de control - Profesional

Crear Control Cardiovascular

Para crear un control del programa de salud cardiovascular, necesita por lo menos llenar los siguientes campos:

Paciente - Fecha de control - Profesional - Peso - Talla - IMC - Diagnóstico nutricional - Presión arterial de pie - Presión arterial sentado - Riesgo cardiovascular
