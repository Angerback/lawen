En esta sección podrá crear y ver atenciones del tipo EMPA.

Para crear atenciones, debe hacer clic en crear.
Para ver atenciones, seleccione la atención a visualizar y haga clic en ver.

Crear atención EMPA

Para crear una atención del tipo EMPA, necesita por lo menos llenar los siguientes campos:

Paciente - Fecha - Profesional.
